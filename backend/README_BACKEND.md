# Backend notes

This bundle contains the project-specific Laravel files for the fragrances.guru starter.

## What to do
1. Create a fresh Laravel project in the `backend` folder.
2. Run `php artisan install:api`.
3. Copy these files into that Laravel project.
4. Run migrations and seeders.
5. Run `php artisan storage:link` so uploaded product images load from `/storage/...`.

## Default seeded admin
- Email: `admin@fragrances.guru`

Change it immediately after first login.
